# JS 知识

## 知识架构

### 目录结构

- JAVASCRIPT
  - 文法
    - 词法
    - 语法
  - 语义
  - 运行时
    - 数据结构
      - 类型
        - 对象
      - 实例
        - 应用和机制
    - 执行过程(算法)
      - 事件循环
      - 微任务的执行
      - 函数的执行
      - 语句级的执行

### 结构的解释

按照编程语言的一般规律：**_用一定的词法和语法，表达一定的语义，从而操作运行时。_**

按照程序的一般规律：**_运行时分为数据结构和算法，数据结构包含类型和实例（JavaScript 的数据类型包含 7 种基本类型和 7 种语言类型，实例就是内置对象的一部分），算法就是指执行过程。_**
